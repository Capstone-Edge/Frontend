# SmartHome AIoT - Frontend

React + TypeScript 기반 스마트홈 AIoT 시뮬레이터 프론트엔드입니다.

## 기술 스택

- **React 18** + **TypeScript**
- **Vite** - 빌드 도구
- **Tailwind CSS** - 스타일링
- **Zustand** - 상태 관리
- **WebSocket** - 실시간 기기 상태 동기화

## 실행 방법

### 1. 패키지 설치

```bash
npm install
```

### 2. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 http://localhost:5173 으로 접근합니다.

> **주의:** 백엔드 서버(http://localhost:8000)가 먼저 실행되어 있어야 합니다.

## 빌드

```bash
npm run build
```
